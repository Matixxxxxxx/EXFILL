# PROJECT EXFIL

# Economy Philosophy

Version: 0.1


# 1. Purpose

The economy creates long-term motivation.

It connects:

- raids;
- loot;
- trading;
- progression;
- crafting.


# 2. Core Philosophy

Money should be useful.

Not just a number.


Players should constantly decide:

Spend?

Save?

Risk?


# 3. Economy Goals


The economy should create:


## Value

Items should matter.


## Decisions

Players choose investments.


## Progression

Players improve over time.


## Risk

Expensive equipment creates tension.


# 4. Avoid


Do not create:

- infinite money generation;
- useless currency;
- one optimal strategy.


# 5. Final Goal

The economy should make every raid meaningful.
