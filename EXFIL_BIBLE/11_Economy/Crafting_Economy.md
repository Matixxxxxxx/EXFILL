# Crafting Economy

Version: 0.1


# 1. Purpose

Crafting creates additional item value.


# 2. Crafting Resources


Examples:


Electronics.


Materials.


Components.


Chemicals.


# 3. Economy Impact


Crafting creates demand for:

- loot;
- resources;
- exploration.


# 4. Rule

Crafting should support economy.

Not replace looting.
