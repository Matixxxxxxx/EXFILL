# PROJECT EXFIL

# Currency System

Version: 0.1


# 1. Purpose

Currency represents economic power.


# 2. Currency Types


## Main Currency

Used for:

- trading;
- equipment;
- services.


## Special Currency

Used for:

- rare items;
- unique upgrades.


## Reputation Currency

Represents relationships.


# 3. Currency Sources


Players obtain currency through:


- extraction;
- selling loot;
- contracts;
- events;
- trading.


# 4. Currency Loss


Money should leave the economy through:


- equipment purchases;
- upgrades;
- services;
- crafting.


# 5. Goal

Currency must constantly move.
