# PROJECT EXFIL

# Economy Balance Rules

Version: 0.1


# 1. Purpose

Maintain healthy progression.


# 2. Rules


Rich players should not become untouchable.


Poor players should have comeback opportunities.


Risk should always have reward.


# 3. Avoid


Do not create:

- impossible grind;
- worthless loot;
- unlimited wealth.


# 4. Final Rule

The economy should create choices.
