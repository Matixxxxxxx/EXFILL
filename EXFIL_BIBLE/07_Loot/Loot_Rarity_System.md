# PROJECT EXFIL

# Loot Rarity System

Version: 0.1


# 1. Purpose

Rarity creates excitement and discovery.


# 2. Rarity Levels


## Common

Frequently found.

Basic value.


## Uncommon

Less frequent.

Useful items.


## Rare

Difficult to find.

High value.


## Very Rare

Special discoveries.


## Legendary

Extremely limited items.

Creates memorable moments.


# 3. Important Rule

Rare does not always mean powerful.


A rare item can be valuable because:

- it is needed;
- it is collectible;
- it has history.


# 4. Goal

Finding rare loot should feel special.
