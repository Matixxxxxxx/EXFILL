# Hidden Locations

Version: 0.1


# 1. Purpose

Hidden locations reward exploration.


# 2. Types


Examples:

- secret rooms;
- abandoned areas;
- underground locations;
- temporary discoveries.


# 3. Design Goal

Players should share discoveries.

Example:

"Did you know there is a hidden room here?"


# 4. Rule

Information itself becomes valuable.
