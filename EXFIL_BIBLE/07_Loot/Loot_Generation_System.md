# PROJECT EXFIL

# Loot Generation System

Version: 0.1


# 1. Purpose

Loot generation creates replayability.


# 2. Loot Should Be Dynamic


The same location should not always contain the same items.


# 3. Generation Factors


Possible factors:


Location:

Buildings have different loot.


Risk:

Dangerous areas have higher chances.


Events:

Temporary opportunities.


Player Economy:

Demand affects value.


# 4. Goal

Players should explore, not memorize everything.
