# Loot Categories

Version: 0.1


# 1. Purpose

Loot categories define the role of items.


# 2. Main Categories


## Equipment Loot

Items used by players.


Examples:

- weapons;
- armor;
- attachments.


## Economic Loot

Items mainly used for trading.


Examples:

- valuables;
- resources;
- rare materials.


## Quest Loot

Items connected to objectives.


Examples:

- documents;
- special objects.


## Crafting Resources

Used for creation and upgrades.


Examples:

- electronics;
- chemicals;
- components.


## Collection Items

Rare items with special value.


# 3. Design Rule

Every category must have a purpose.
