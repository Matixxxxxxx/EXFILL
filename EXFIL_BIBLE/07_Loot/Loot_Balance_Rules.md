# PROJECT EXFIL

# Loot Balance Rules

Version: 0.1


# 1. Purpose

Prevent loot from becoming meaningless.


# 2. Rules


Rare loot:

Must be exciting.


Common loot:

Must still have value.


High value areas:

Must create danger.


# 3. Avoid


Do not create:

- guaranteed farming spots;
- boring loot routes;
- useless items.


# 4. Final Rule

Loot should create decisions, not just rewards.
