# PROJECT EXFIL

# Loot Distribution System

Version: 0.1


# 1. Purpose

Loot distribution creates different risk zones.


# 2. Location Value


Low Risk Areas:

- common loot;
- safer exploration.


Medium Risk Areas:

- better loot;
- more enemies.


High Risk Areas:

- rare loot;
- strong threats;
- high PvP activity.


# 3. Important Rule

The best loot should require effort.


# 4. Goal

Players decide:

Safe profit?

Or dangerous opportunity?
