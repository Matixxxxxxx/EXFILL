# PROJECT EXFIL

# Loot Philosophy

Version: 0.1


# 1. Purpose

Loot creates motivation for exploration and risk.


Loot is not only a reward.

Loot creates decisions.


# 2. Core Question

Every valuable item should create a choice:


"Do I leave now or continue?"


# 3. Loot Should Create Stories


Example:

A player finds a rare weapon component.

The extraction is far away.

Other players may be nearby.


The decision creates the memory.


# 4. Loot Value Sources


Value comes from:

- rarity;
- usefulness;
- demand;
- danger required to obtain.


# 5. Goal

Loot should make every raid feel different.
