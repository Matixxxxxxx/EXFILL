# PROJECT EXFIL

# PvE Combat

Version: 0.1


# 1. Purpose

AI enemies create:

- atmosphere;
- pressure;
- additional danger.


PvE exists to improve the raid experience.


# 2. AI Combat Goals


AI should:

- react to players;
- protect areas;
- create tactical situations.


AI should not:

- replace PvP;
- feel unfair;
- behave randomly.


# 3. AI Difficulty


Different locations can have different threats.


Low danger:

- basic enemies;
- predictable behavior.


High danger:

- stronger enemies;
- better equipment;
- tactical behavior.


# 4. Example Situation


A squad enters a building.

AI detects them.

The fight creates noise.

Another squad hears the fight.

The situation changes.


# 5. Goal

AI should create opportunities and problems.
