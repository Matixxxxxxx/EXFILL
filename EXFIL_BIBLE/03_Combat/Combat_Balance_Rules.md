# Combat Balance Rules

Version: 0.1


# 1. Purpose

These rules guide combat balancing.


# 2. Equipment Advantage

Better equipment should provide:

advantage.

Not:

guaranteed victory.


# 3. Skill Importance

Skill should affect:

- aiming;
- positioning;
- decision making;
- teamwork.


# 4. Time To Kill

Combat should be:

fast enough to feel dangerous;

slow enough to allow decisions.


# 5. Weapon Diversity

Every weapon category needs:

- purpose;
- situation;
- strengths;
- weaknesses.


# 6. Balance Testing

Every combat change should consider:

- beginner players;
- experienced players;
- solo players;
- squads.


# 7. Final Rule

A player should lose because of a mistake.

Not because the game decided the outcome before the fight started.
