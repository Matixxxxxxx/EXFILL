# Damage System

Version: 0.1


# 1. Purpose

Damage should make combat meaningful.

A hit should matter.


# 2. Damage Factors


Damage depends on:


## Hit Location

Different body parts have different consequences.


## Ammunition

Different ammunition types behave differently.


## Armor

Protection changes survival chances.


## Distance

Weapon effectiveness changes with range.


# 3. Goal

A fight should not be decided only by equipment.


A prepared player with better decisions can win.


# 4. Important Rule

Avoid:

Instant deaths without reaction.

Combat should allow:

- counterplay;
- decisions;
- adaptation.
