# PROJECT EXFIL

# Gunplay Design

Version: 0.1


# 1. Purpose

Gunplay defines how weapons feel in the hands of players.


# 2. Main Goals


Weapons should feel:

- different;
- powerful;
- understandable;
- controllable.


# 3. Weapon Identity

Every weapon should have:

Strengths:

What makes it useful?


Weaknesses:

What limits it?


Example:


Assault Rifle:

+

- versatile;
- reliable.

-

- expensive ammunition.


Shotgun:

+

- powerful close range.

-

- weak at distance.


# 4. Skill Expression

Gunfights should reward:

- aiming;
- recoil control;
- positioning;
- timing.


# 5. Avoid

Do not create:

- automatic weapons that dominate everything;
- weapons with no purpose;
- artificial randomness.
