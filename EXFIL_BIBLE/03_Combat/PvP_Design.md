# PROJECT EXFIL

# PvP Design

Version: 0.1


# 1. Purpose

PvP is one of the main sources of memorable moments in EXFIL.

Player fights should create:

- tension;
- excitement;
- competition;
- stories.


# 2. PvP Philosophy

The goal is not to create constant fighting.

The goal is to create meaningful encounters.


A fight should happen because:

- players made a choice;
- players wanted a reward;
- players had different goals.


# 3. Types of Encounters


## Planned Fight

Players intentionally engage.

Example:

A squad attacks a high-value location.


## Unexpected Encounter

Players discover each other.

Example:

Two teams enter the same building.


## Defensive Fight

Players protect:

- loot;
- objective;
- extraction point.


# 4. Strong Player vs Weak Player

Equipment advantage should exist.

However:

A better equipped player should not have guaranteed victory.


Skill factors:

- positioning;
- awareness;
- teamwork;
- tactics.


# 5. PvP Goal

Every player fight should create a story.
