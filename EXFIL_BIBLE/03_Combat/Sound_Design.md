# PROJECT EXFIL

# Sound Design

Version: 0.1


# 1. Purpose

Sound is information.


Players should be able to understand:

- danger;
- distance;
- direction.


# 2. Important Sounds


Examples:

- footsteps;
- gunshots;
- reloads;
- doors;
- equipment movement.


# 3. Gameplay Role

Sound should allow players to think:


"I heard something."


Not:


"A random sound happened."


# 4. Goal

Audio creates awareness and tension.
