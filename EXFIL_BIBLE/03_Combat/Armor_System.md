# PROJECT EXFIL

# Armor System

Version: 0.1


# 1. Purpose

Armor creates strategic choices.


Armor is not:

"More armor = guaranteed victory."


# 2. Armor Effects


Armor influences:

- survival;
- movement;
- cost;
- risk.


# 3. Player Decision


Heavy armor:


+

better protection.


-

more expensive.

-

more noticeable.


Light armor:


+

mobility.

+

lower risk.


-

less protection.


# 4. Design Goal

Armor should change decisions before and during combat.
