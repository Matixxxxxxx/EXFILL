# PROJECT EXFIL

# Movement System

Version: 0.1


# 1. Purpose

Movement defines tactical possibilities.


# 2. Movement Features


Possible systems:

- sprint;
- crouch;
- prone;
- leaning;
- vaulting;
- stamina.


# 3. Design Goals


Movement should reward:

- positioning;
- awareness;
- planning.


# 4. Avoid

Do not make movement:

- unrealistic arcade;
- slow and frustrating.


# 5. Goal

Movement should feel responsive but tactical.
