# PROJECT EXFIL

# Project Scope

Version: 0.1


# 1. Project Goal

Create a high-quality extraction shooter experience on Roblox.

The focus is:

- tactical combat;
- meaningful raids;
- progression;
- teamwork;
- memorable player stories.


# 2. Core Features

The planned foundation includes:


## Extraction Gameplay

Players enter dangerous locations, collect resources and attempt extraction.


## PvPvE Combat

Players fight:

- other players;
- AI enemies;
- special threats.


## Equipment System

Players manage:

- weapons;
- armor;
- medical supplies;
- tactical equipment.


## Economy

Players use resources for:

- trading;
- upgrades;
- progression.


## Long-Term Progression

Players develop:

- account progression;
- hideout;
- reputation;
- collections.


# 3. Non Goals

EXFIL will not focus on:

- becoming a military simulator;
- unlimited complexity;
- realism without gameplay value;
- random content quantity.


# 4. Development Philosophy

First create:

A fun and stable core.


Then expand:

More weapons.
More maps.
More systems.


Quality before quantity.
