# PROJECT EXFIL

# Glossary

Version: 0.1


# Extraction Shooter

A game genre where players enter a dangerous area, collect resources and attempt to leave alive.


# Raid

One complete gameplay session inside a map.


# Extraction

The process of leaving the raid successfully.


# PvP

Player versus Player combat.


# PvE

Player versus Environment combat against AI.


# PvPvE

A combination of player combat and AI threats.


# Loot

Items collected during raids.


# Risk

Potential loss or danger involved in a decision.


# Reward

Potential benefit gained from successful actions.


# Progression

Long-term player development.


# Meta

The larger game systems outside individual raids.

Examples:

- economy;
- reputation;
- hideout.
