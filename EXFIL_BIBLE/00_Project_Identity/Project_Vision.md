# PROJECT EXFIL

# Project Vision

Version: 0.1

Status: Foundation Phase


# 1. Project Overview

EXFIL is a realistic PvPvE Extraction Shooter designed for Roblox.

The project is inspired by games such as:

- Arena Breakout Infinite
- Escape from Tarkov
- Hunt: Showdown
- DayZ

However, EXFIL is not designed to copy existing games.

The goal is to create its own identity by focusing on:

- memorable raids;
- meaningful decisions;
- tactical gameplay;
- player-created stories.


# 2. Core Vision

The main idea of EXFIL:

> Every raid should become a story.


Players should not remember only:

- how much money they earned;
- what weapon they unlocked;
- what level they reached.


They should remember:

- the fight they survived;
- the decision they made;
- the teammate they saved;
- the impossible victory they achieved.


Example:

"Two players with basic equipment defeated a fully geared squad because they used better positioning and teamwork."


# 3. Game Identity

EXFIL is:

A tactical extraction shooter where every decision has consequences.


The player experience should be based on:

## Preparation

Choosing:
- weapons;
- armor;
- equipment;
- strategy.


## Risk

Every raid contains a choice:

Continue searching for better rewards?

Or leave safely?


## Consequences

Victory and failure should both matter.


# 4. Design Goal

The goal is not to create:

- the biggest game;
- the most weapons;
- the most maps.


The goal is to create:

> The most memorable extraction experience on Roblox.


# 5. Main Design Question

Before adding any feature, developers must ask:


"Does this create interesting decisions or memorable moments?"


If the answer is no:

The feature should be reconsidered.


# 6. Long-Term Vision

EXFIL should become a game where players return because:

- every raid can create a new story;
- improving skill matters;
- teamwork matters;
- knowledge matters.


The player should always think:

"What will happen in my next raid?"
