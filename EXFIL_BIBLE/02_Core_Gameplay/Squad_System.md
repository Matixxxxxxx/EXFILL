# PROJECT EXFIL

# Squad System

Version: 0.1


# 1. Purpose

Squads create some of the most memorable moments in extraction shooters.

Teamwork should create advantages without removing individual skill.


# 2. Squad Size

Target:

Maximum:
4 players per squad.


Reason:

Small squads create:

- communication;
- responsibility;
- tactical decisions.


# 3. Squad Roles

EXFIL should not use strict classes.

Players are flexible.


Possible temporary roles:


## Scout

Focus:

- information;
- positioning;
- observation.


## Assault

Focus:

- direct combat.


## Support

Focus:

- resources;
- healing;
- equipment management.


## Specialist

Focus:

- technical tasks;
- objectives.


# 4. Team Decisions


Examples:


"Do we help our teammate or escape?"


"Do we attack this squad?"


"Who carries the valuable loot?"


# 5. Squad Death


A fallen teammate creates tension.


The team must decide:

- recover equipment;
- continue mission;
- abandon area.


# 6. Goal

Players should remember:

"We survived together."
