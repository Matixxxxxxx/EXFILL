# PROJECT EXFIL

# Long Term Progression

Version: 0.1


# 1. Purpose

Progression gives players long-term goals.


However:

Progression should support gameplay.

Not replace it.


# 2. Progression Categories


## Character Development

Examples:

- skills;
- reputation;
- achievements.


## Equipment Development

Examples:

- better gear;
- modifications;
- collections.


## Personal Development

Examples:

- hideout;
- statistics;
- personal milestones.


# 3. Avoid


Do not create progression based only on:

- grinding;
- repetitive tasks;
- artificial numbers.


# 4. End Goal


A veteran player should be valuable because of:

- knowledge;
- skill;
- experience.


Not only because they played longer.
