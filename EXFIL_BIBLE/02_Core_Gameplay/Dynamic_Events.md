# PROJECT EXFIL

# Dynamic Events

Version: 0.1


# 1. Purpose

Dynamic events make raids unpredictable.


The goal:

The same map should create different stories.


# 2. Event Types


## Combat Events

Examples:

- enemy patrol;
- AI reinforcement;
- special target.


## Loot Events

Examples:

- supply drop;
- valuable container;
- rare location activation.


## World Events

Examples:

- power failure;
- weather change;
- emergency situation.


# 3. Rules


Events should:

- create decisions;
- increase tension;
- reward exploration.


Events should not:

- randomly kill players;
- remove player control.


# 4. Frequency


Events should feel special.

Too many events reduce importance.
