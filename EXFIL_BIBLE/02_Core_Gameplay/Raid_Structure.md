# PROJECT EXFIL

# Raid Structure

Version: 0.1


# 1. Overview

A raid is the main gameplay session in EXFIL.


A raid combines:

- exploration;
- combat;
- risk;
- progression.


# 2. Raid Phases


# Phase 1 — Preparation


Before entering:

Player chooses:

- weapons;
- armor;
- medical supplies;
- tactical equipment.


The player decides:

Low risk:

cheap equipment.

High risk:

expensive equipment.


---

# Phase 2 — Infiltration


The player enters the map.


Important elements:

- unknown situation;
- limited information;
- possible threats.


Goal:

Create tension from the beginning.


---

# Phase 3 — Operation


The main part of the raid.


Possible activities:

- searching buildings;
- fighting enemies;
- completing tasks;
- finding rare items.


---

# Phase 4 — Extraction


The player decides:

Leave with current rewards.

OR

Continue searching.


This decision creates the main extraction tension.


---

# 3. Raid Duration


Target:

Small maps:

25-35 minutes.


Medium maps:

35-45 minutes.


Large maps:

45+ minutes.


Duration depends on:

- map size;
- objectives;
- player decisions.
