# PROJECT EXFIL

# Extraction System

Version: 0.1


# 1. Purpose

Extraction is the emotional climax of every raid.


# 2. Extraction Should Feel Dangerous


The player should feel:


"I am almost safe, but not yet."


# 3. Extraction Decisions


The player chooses:


Leave now:

+
secure rewards

-
miss possible opportunities


Continue:

+
more rewards

-
higher danger


# 4. Successful Extraction


Creates:

- satisfaction;
- progression;
- memorable moments.


# 5. Failed Extraction


Creates:

- learning;
- adaptation;
- motivation.
