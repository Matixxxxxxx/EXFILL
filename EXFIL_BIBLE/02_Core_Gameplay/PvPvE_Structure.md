# PROJECT EXFIL

# PvPvE Structure

Version: 0.1


# 1. Overview

EXFIL combines:

Player versus Player

and

Player versus Environment.


Both threats should matter.


# 2. PvP Purpose


PvP creates:

- unpredictable situations;
- emotional victories;
- tactical encounters.


# 3. PvE Purpose


AI creates:

- pressure;
- atmosphere;
- additional danger.


PvE should not replace PvP.

It should create situations around PvP.


# 4. Example Situation


A squad enters a building.

They hear AI enemies.

While fighting:

another player squad arrives.


The situation changes instantly.


# 5. Design Goal


Players should never know:

"Is the biggest threat a player or the environment?"
