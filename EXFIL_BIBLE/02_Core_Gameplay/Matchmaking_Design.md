# PROJECT EXFIL

# Matchmaking Design

Version: 0.1


# 1. Purpose

Matchmaking creates fair and exciting raids.


# 2. Squad Structure


Target:

4 players maximum per squad.


Raid:

Multiple squads in one location.


Goal:

Create uncertainty.


# 3. Player Count


The final number depends on map size.


Examples:


Small map:

fewer players.

High intensity.


Large map:

more players.

More exploration.


# 4. Balance Factors


Matchmaking should consider:

- squad size;
- player experience;
- map selection.


# 5. Important Rule


Perfect fairness is less important than creating exciting situations.

Extraction shooters need uncertainty.
