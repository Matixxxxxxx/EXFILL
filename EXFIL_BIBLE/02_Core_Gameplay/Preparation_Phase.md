# PROJECT EXFIL

# Preparation Phase

Version: 0.1


# 1. Purpose

Preparation creates emotional value before the raid begins.


A player should think:


"What am I willing to risk?"


# 2. Equipment Choice


Every item represents a decision.


Example:


Cheap weapon:

+
low risk

-
lower effectiveness


Expensive weapon:

+
better performance

-
higher emotional loss


# 3. Loadout Philosophy


A loadout should represent:

- playstyle;
- strategy;
- confidence.


Players should not have only one optimal choice.


# 4. Preparation Is Gameplay


The menu before the raid is part of the experience.

Planning should matter.
