# PROJECT EXFIL

# Death and Loss

Version: 0.1


# 1. Purpose

Loss creates meaning.

Without risk, success has no value.


# 2. What Can Be Lost


Possible losses:

- weapons;
- armor;
- ammunition;
- loot;
- consumables.


# 3. What Should Remain


Long-term progress should protect player motivation.


Examples:

- achievements;
- cosmetics;
- account progression;
- unlocked knowledge.


# 4. Death Experience


After death the player should think:


"I made a mistake."


Not:


"The game wasted my time."


# 5. Balance

Loss should create tension, not punishment.
