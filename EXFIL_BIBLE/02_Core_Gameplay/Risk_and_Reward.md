# Risk and Reward System

Version: 0.1


# 1. Core Principle

The bigger the reward, the greater the danger.


# 2. Risk Sources


Risk can come from:


## Location

Dangerous areas contain better rewards.


## Equipment

Expensive gear creates bigger losses.


## Time

The longer the player stays:

- more loot can be found;
- more danger appears.


## Information

Unknown situations create uncertainty.


# 3. Reward Types


Rewards include:

- currency;
- equipment;
- rare items;
- progression;
- reputation.


# 4. Important Rule


The safest option should not always be the best.


Players should constantly evaluate:

"Is this worth the risk?"
