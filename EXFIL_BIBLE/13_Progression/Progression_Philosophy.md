# Progression Philosophy

Version: 0.1


# 1. Purpose

Progression creates long-term player motivation.


It rewards:

- experience;
- dedication;
- achievements;
- mastery.


# 2. Core Philosophy

Progression should expand possibilities.

Not create unfair advantages.


# 3. Progression Sources


Players progress through:


- successful extractions;
- quests;
- discoveries;
- achievements;
- combat experience;
- exploration.


# 4. Avoid


Do not create:

- pure grinding;
- impossible requirements;
- pay-to-win advantages.


# 5. Final Goal

Every player should feel improvement.
