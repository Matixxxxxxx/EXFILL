# PROJECT EXFIL

# Prestige System

Version: 0.1


# 1. Purpose

Prestige creates an endless long-term goal.


# 2. Concept


After reaching maximum progression:

Player can reset part of progress.


In exchange receives:

- exclusive rewards;
- status;
- cosmetics;
- unique items.


# 3. Reset Rules


Prestige may reset:


- level;
- trader progression;
- some upgrades.


Player keeps:


- achievements;
- cosmetics;
- prestige rewards;
- rare collections.


# 4. Goal

Experienced players receive recognition.

Not permanent power.
