# PROJECT EXFIL

# Season System

Version: 0.1


# 1. Purpose

Seasons create fresh goals.


# 2. Philosophy

Seasons should add content.

Not delete player history.


# 3. Season Content


Possible additions:


- events;
- new locations;
- quests;
- cosmetics;
- challenges.


# 4. Avoid


Do not completely wipe progress.


# 5. Goal

Players return because the world evolves.
