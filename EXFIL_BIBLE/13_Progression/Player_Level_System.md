# PROJECT EXFIL

# Player Level System

Version: 0.1


# 1. Purpose

Player level represents experience.


# 2. Level Sources


Experience can be gained from:


- extraction;
- missions;
- discovering locations;
- defeating enemies;
- completing objectives.


# 3. Level Benefits


Level can unlock:


- access to traders;
- new quests;
- hideout options;
- cosmetics.


# 4. Important Rule

Level should not directly determine combat power.


# 5. Goal

Level shows experience, not superiority.
