# Skill System

Version: 0.1


# 1. Purpose

Skills represent player specialization.


# 2. Skill Philosophy

Skills should improve efficiency.

Not create impossible advantages.


# 3. Skill Categories


Possible categories:


## Combat

Examples:

- weapon handling;
- recoil control;
- tactical movement.


## Survival

Examples:

- inventory efficiency;
- resource management.


## Technical

Examples:

- crafting;
- equipment maintenance.


## Exploration

Examples:

- navigation;
- information gathering.


# 4. Rule

Skills should reward playstyle.
