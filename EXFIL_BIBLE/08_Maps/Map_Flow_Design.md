# PROJECT EXFIL

# Map Flow Design

Version: 0.1


# 1. Purpose

Map flow controls player movement.


# 2. Good Flow Creates


- encounters;
- decisions;
- tension.


# 3. Movement Routes


Every area should have:


Main route:

Fast but dangerous.


Alternative route:

Slower but safer.


Hidden route:

Requires knowledge.


# 4. Avoid

Do not create:

- forced paths;
- dead zones;
- predictable movement.


# 5. Goal

Players should constantly make decisions.
