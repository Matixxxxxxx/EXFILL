# PROJECT EXFIL

# Map Philosophy

Version: 0.1


# 1. Purpose

Maps are the foundation of the extraction experience.


A map creates:

- exploration;
- combat situations;
- risk decisions;
- memorable moments.


# 2. Core Philosophy

A good map is not just beautiful.

A good map creates gameplay.


# 3. Map Design Goals


Every map should provide:


## Multiple Strategies

Players should have choices.


Examples:

- aggressive route;
- safe route;
- stealth route.


## Risk And Reward

Dangerous areas should provide better opportunities.


## Replayability

Players should discover something new.


# 4. Avoid

Do not create:

- empty spaces;
- only one optimal route;
- repetitive locations.


# 5. Final Goal

Every raid should feel like a unique story.
