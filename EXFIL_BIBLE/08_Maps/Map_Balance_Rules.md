# PROJECT EXFIL

# Map Balance Rules

Version: 0.1


# 1. Purpose

Maintain fair and interesting maps.


# 2. Rules


No area should be:

- completely useless;
- impossible to survive;
- mandatory every raid.


# 3. Important Factors


Consider:

- loot;
- movement;
- combat;
- extraction;
- player population.


# 4. Final Rule

The best map is not where players fight the most.

The best map is where players constantly make meaningful choices.
