# PROJECT EXFIL

# Point Of Interest System

Version: 0.1


# 1. Purpose

POI create reasons to move around the map.


# 2. POI Types


## High Value Location

Contains:

- rare loot;
- bosses;
- special objectives.


High risk.


---

## Medium Value Location

Contains:

- useful loot;
- normal combat.


---

## Low Value Location

Contains:

- basic resources;
- safer exploration.


# 3. POI Requirements


Every important location needs:


- identity;
- gameplay purpose;
- reason to visit.


# 4. Goal

Players should remember locations.
