# PROJECT EXFIL

# Map Types

Version: 0.1


# 1. Purpose

Different map types create different gameplay.


# 2. Main Map Categories


## Urban Map

Example:

Destroyed city.

Gameplay:

- buildings;
- close combat;
- ambushes.


Strength:

High tension.


Weakness:

Complex navigation.



---

## Industrial Map

Example:

Factory, port, warehouses.


Gameplay:

- medium range fights;
- open areas;
- vertical gameplay.



---

## Laboratory Map

Example:

Secret research facility.


Gameplay:

- high value loot;
- dangerous enemies;
- close combat.



---

## Wilderness Map

Example:

Forest, mountains, valley.


Gameplay:

- long distance fights;
- exploration;
- survival.



---

## Special Event Map

Temporary locations.

Used for:

- events;
- bosses;
- unique rewards.


# 3. Goal

Every map type should create a different emotion.
