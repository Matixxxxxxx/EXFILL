# Map Size System

Version: 0.1


# 1. Purpose

Map size affects raid pacing.


# 2. Map Categories


## Small Maps


Raid length:

20-30 minutes.


Characteristics:

- intense fights;
- fast extraction;
- limited exploration.



## Medium Maps


Raid length:

30-45 minutes.


Characteristics:

- balanced gameplay;
- multiple routes.



## Large Maps


Raid length:

45-60+ minutes.


Characteristics:

- exploration;
- long distance combat;
- complex decisions.


# 3. Design Rule

Size depends on gameplay purpose.

Bigger is not always better.
