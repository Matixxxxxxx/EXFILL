# PROJECT EXFIL

# Hideout Philosophy

Version: 0.1


# 1. Purpose

Hideout is the player's personal progression space.

It represents:

- survival;
- preparation;
- achievements;
- long-term investment.


# 2. Core Philosophy

Hideout should create attachment.

Players should feel:

"This is my base."


# 3. Hideout Functions


Hideout provides:


## Preparation

Before raids:

- equipment management;
- planning;
- customization.


## Progression

Players improve their capabilities.


## Personal Identity

Players customize their space.


# 4. Avoid


Do not create:

- meaningless upgrades;
- pure resource sinks;
- mandatory grinding.


# 5. Final Goal

Hideout should make players want to return.
