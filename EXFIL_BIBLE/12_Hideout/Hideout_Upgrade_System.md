# PROJECT EXFIL

# Hideout Upgrade System

Version: 0.1


# 1. Purpose

Upgrades improve hideout capabilities.


# 2. Upgrade Structure


Each module has levels.


Example:


Workshop Level 1:

Basic modifications.


Workshop Level 2:

Advanced modifications.


Workshop Level 3:

Rare modifications.


# 3. Upgrade Requirements


Possible requirements:


- currency;
- resources;
- quest completion;
- rare items.


# 4. Design Rule

Higher levels should unlock possibilities.

Not only bigger numbers.
