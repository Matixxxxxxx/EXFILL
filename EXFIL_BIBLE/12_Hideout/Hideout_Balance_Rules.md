# Hideout Balance Rules

Version: 0.1


# 1. Purpose

Prevent hideout from becoming boring progression.


# 2. Rules


Good upgrades:

- unlock features;
- create choices;
- improve experience.


Bad upgrades:

- only increase numbers;
- require endless grinding.


# 3. Final Rule

Hideout should reward dedication.

Not punish casual players.
