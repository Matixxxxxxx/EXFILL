# PROJECT EXFIL

# Helmet System

Version: 0.1


# 1. Purpose

Helmets protect critical areas and create risk decisions.


# 2. Helmet Features


Possible protection:

- head;
- face;
- ears.


# 3. Helmet Tradeoffs


Heavy helmet:

+
better protection

-
reduced mobility
-
higher cost


Light helmet:

+
mobility

-
lower protection


# 4. Face Protection

Face protection is valuable because:

- it protects vulnerable areas;
- changes combat outcomes.


# 5. Design Goal

Choosing a helmet should depend on:

- mission;
- budget;
- expected threats.
