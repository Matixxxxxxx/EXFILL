# PROJECT EXFIL

# Equipment Philosophy

Version: 0.1


# 1. Purpose

Equipment defines player preparation before a raid.

Every item represents:

- protection;
- capability;
- risk;
- investment.


# 2. Core Philosophy

Equipment should create decisions.


The player should think:

"Is this worth bringing?"


Not:

"What is the strongest setup?"


# 3. Equipment Categories


Main categories:

- armor;
- helmets;
- tactical rigs;
- backpacks;
- medical equipment;
- tactical devices;
- special equipment.


# 4. Equipment Has A Cost

Every advantage has a tradeoff.


Examples:


Heavy armor:

+
high protection

-
high cost
-
lower mobility


Large backpack:

+
more loot capacity

-
larger target
-
more weight


# 5. Design Goal

Equipment should influence strategy, not remove skill.
