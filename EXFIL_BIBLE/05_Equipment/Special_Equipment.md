# PROJECT EXFIL

# Special Equipment

Version: 0.1


# 1. Purpose

Special equipment creates unique gameplay opportunities.


# 2. Examples


Night Vision:

Allows operations in darkness.


Thermal Devices:

Provides information advantage.


Breaching Tools:

Allows alternative routes.


Electronic Equipment:

Interacts with systems.


# 3. Balance

Special equipment should be:

- valuable;
- limited;
- situational.


# 4. Goal

Create different ways to approach a raid.
