# PROJECT EXFIL

# Tactical Rig System

Version: 0.1


# 1. Purpose

Tactical rigs define how much equipment a player can carry.


# 2. Functions


A rig provides:

- magazine storage;
- quick access slots;
- equipment organization.


# 3. Rig Types


## Light Rig

Purpose:

Fast movement.

Limited storage.


## Combat Rig

Purpose:

Balanced combat.


## Heavy Rig

Purpose:

Maximum equipment.

Reduced mobility.


# 4. Design Goal

Rig choice should match playstyle.
