# PROJECT EXFIL

# Quest Philosophy

Version: 0.1


# 1. Purpose

Quests create goals and direction for players.


They provide:

- motivation;
- progression;
- world interaction;
- rewards.


# 2. Core Philosophy

Quests should create situations.

Not simple tasks.


# 3. Good Quest Example


Objective:

Recover lost military data.


Location:

Abandoned communication center.


Risk:

Enemy faction controls area.


Reward:

Access to new information.


# 4. Avoid


Do not create:

- repetitive farming;
- boring checklist tasks;
- meaningless objectives.


# 5. Final Goal

Every quest should create a memorable experience.
