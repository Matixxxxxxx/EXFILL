# PROJECT EXFIL

# Dynamic Quest System

Version: 0.1


# 1. Purpose

Dynamic quests make the world feel alive.


# 2. Sources


Dynamic quests can appear from:


- player actions;
- faction conflicts;
- world events;
- economy changes.


# 3. Examples


Power station damaged.

New objective:

Restore power equipment.


Faction loses territory.

New objective:

Help recover location.


# 4. Goal

The world creates opportunities.
