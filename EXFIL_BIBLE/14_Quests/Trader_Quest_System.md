# PROJECT EXFIL

# Trader Quest System

Version: 0.1


# 1. Purpose

Trader quests create relationships with NPCs.


# 2. Trader Goals


Different traders have different interests.


Examples:


Weapons Trader:

Needs weapon parts.


Medical Trader:

Needs medical supplies.


Information Trader:

Needs documents.


# 3. Rewards


Trader quests provide:


- reputation;
- access;
- discounts;
- unique items.


# 4. Goal

Players choose which relationships to develop.
