# PROJECT EXFIL

# Quest Balance Rules

Version: 0.1


# 1. Purpose

Maintain interesting quests.


# 2. Rules


Good quests:

- create decisions;
- involve the world;
- reward exploration.


Bad quests:

- repetitive farming;
- forced grinding;
- boring objectives.


# 3. Final Rule

A quest should create a story, not a chore.
