# Quest Sharing System

Version: 0.1


# 1. Purpose

Supports team gameplay.


# 2. Features


Players can:

- share objectives;
- complete missions together;
- divide roles.


# 3. Team Examples


One player:

Carries objective item.


Another:

Provides protection.


Another:

Searches information.


# 4. Goal

Quests should encourage cooperation.
