# PROJECT EXFIL

# Inventory Philosophy

Version: 0.1


# 1. Purpose

Inventory is a decision system.

It controls:

- what players bring;
- what players collect;
- what players risk losing.


# 2. Core Philosophy

Inventory space creates value.


If players can carry everything:

- choices disappear;
- loot becomes meaningless.


# 3. Player Decisions


During preparation:

"What equipment should I bring?"


During raid:

"What loot is worth taking?"


After finding valuable items:

"What should I replace?"


# 4. Inventory Should Create Emotion


Example:

A player finds a rare item.

The backpack is full.


The player must decide:

Drop current loot?

Risk staying longer?

Leave?


# 5. Goal

Inventory should create memorable decisions.
