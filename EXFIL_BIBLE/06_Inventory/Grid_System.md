# PROJECT EXFIL

# Grid Inventory System

Version: 0.1


# 1. Purpose

Grid inventory creates physical space limitations.


# 2. Item Size

Every item has:

- width;
- height.


Example:


Small item:

1x1


Weapon:

2x6


Large equipment:

5x8


# 3. Player Decisions


A large valuable item may require:

- more space;
- removing other items.


# 4. Goal

Inventory management becomes gameplay.
