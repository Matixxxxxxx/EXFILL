# Inventory Balance Rules

Version: 0.1


# 1. Purpose

Prevent inventory from becoming a simple storage system.


# 2. Rules


More space:

=
more potential profit.


More weight:

=
more disadvantages.


More protection:

=
higher investment.


# 3. Important Rule

Every inventory advantage must have a cost.


# 4. Goal

Inventory decisions should create stories.
