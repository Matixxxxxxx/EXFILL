# PROJECT EXFIL

# Item System

Version: 0.1


# 1. Purpose

Items are the foundation of inventory and economy.


# 2. Item Categories


## Equipment

Used by players.


Examples:

- weapons;
- armor;
- tools.


## Consumables

Used during gameplay.


Examples:

- medicine;
- ammunition.


## Resources

Used for:

- crafting;
- trading;
- upgrades.


## Valuable Items

High economic value.


# 3. Item Data


Every item should have:


- name;
- category;
- rarity;
- size;
- weight;
- value;
- description.


# 4. Goal

Every item should have a purpose.
