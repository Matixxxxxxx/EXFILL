# PROJECT EXFIL

# Inventory UI UX

Version: 0.1


# 1. Purpose

Inventory interface must be clear and fast.


# 2. Requirements


The player should easily understand:

- what they have;
- what fits;
- what is valuable.


# 3. UI Principles


Avoid:

- unnecessary animations;
- confusing menus;
- slow interactions.


# 4. Goal

Inventory management should feel satisfying.
