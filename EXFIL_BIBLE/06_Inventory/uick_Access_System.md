# PROJECT EXFIL

# Quick Access System

Version: 0.1


# 1. Purpose

Quick access allows players to react quickly during combat.


# 2. Quick Slots


Possible slots:

- weapons;
- medicine;
- grenades;
- tools.


# 3. Design Goals


Quick access should:

- reward preparation;
- reduce unnecessary menu usage;
- improve combat flow.


# 4. Limitation

Players should still choose what is available quickly.
