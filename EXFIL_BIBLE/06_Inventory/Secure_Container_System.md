# PROJECT EXFIL

# Secure Container System

Version: 0.1


# 1. Purpose

Secure containers protect limited valuable items.


# 2. Philosophy

Secure storage reduces frustration.

It should not remove risk completely.


# 3. Rules


Secure containers:

- have limited capacity;
- cannot store everything;
- are valuable progression rewards.


# 4. Design Goal

Players always have:

some protection.

But raids still matter.
