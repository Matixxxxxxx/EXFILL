# PROJECT EXFIL

# Weight System

Version: 0.1


# 1. Purpose

Weight creates realistic equipment decisions.


# 2. Weight Effects


Low weight:

+
fast movement

+
better stamina


High weight:

+
more equipment

-
slower movement

-
higher noise


# 3. Balance Rule

Weight should influence decisions.

It should not make carrying equipment impossible.


# 4. Goal

Players choose between:

mobility

and

capacity.
