# Inventory Structure

# 1. Purpose

Inventory structure defines how players organize equipment.


# 2. Main Sections


## Character Equipment

Currently equipped items.


Includes:

- weapon slots;
- armor;
- helmet;
- rig;
- backpack.


---

## Tactical Slots

Quick access items.


Examples:

- grenades;
- medicine;
- tools.


---

## Backpack Storage

Main loot storage.


---

## Secure Storage

Protected items that survive failure.
