# PROJECT EXFIL

# AI Cover System

Version: 0.1


# 1. Purpose

Cover makes AI combat more realistic.


# 2. Cover Types


Hard Cover:

Blocks bullets.


Soft Cover:

Provides limited protection.


Tactical Cover:

Provides position advantage.


# 3. AI Usage


AI should:

- move between covers;
- avoid open areas;
- create crossfire.


# 4. Goal

AI should look tactical.
