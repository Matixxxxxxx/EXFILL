# PROJECT EXFIL

# AI Squad System

Version: 0.1


# 1. Purpose

Groups of AI should behave together.


# 2. Squad Roles


Possible roles:


Leader:

Makes decisions.


Assault:

Pushes enemies.


Support:

Provides covering fire.


Scout:

Searches information.


# 3. Squad Behavior


A squad can:

- patrol;
- defend;
- attack;
- retreat.


# 4. Goal

Groups should feel organized.
