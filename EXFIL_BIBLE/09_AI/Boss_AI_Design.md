# PROJECT EXFIL

# Boss AI Design

Version: 0.1


# 1. Purpose

Bosses create unique raid experiences.


# 2. Boss Philosophy

Boss should not be:

"Enemy with more health."


Boss should have:

- identity;
- tactics;
- unique behavior.


# 3. Boss Features


Examples:

- special abilities;
- unique equipment;
- guards;
- changing tactics.


# 4. Goal

Players should remember defeating bosses.
