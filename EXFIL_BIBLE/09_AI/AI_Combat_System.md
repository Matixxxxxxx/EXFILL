# AI Combat System

Version: 0.1


# 1. Purpose

Defines how AI fights players.


# 2. Combat Principles


AI should:

- seek advantage;
- use environment;
- coordinate.


# 3. Combat Actions


Possible actions:


Attack:

Direct engagement.


Suppress:

Prevent movement.


Flank:

Create advantage.


Retreat:

Avoid losing fight.


Call Support:

Increase threat.


# 4. Important Rule

AI should win through decisions.

Not through impossible accuracy.
