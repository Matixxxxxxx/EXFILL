# AI Architecture

# 1. Purpose

Defines the structure of all AI systems.


# 2. AI Components


Every AI character consists of:


## Perception System

How AI receives information.


Examples:

- vision;
- hearing;
- damage detection.


---

## Decision System

How AI chooses actions.


Examples:

- attack;
- retreat;
- search;
- defend.


---

## Movement System

Controls navigation.


Examples:

- patrol;
- chase;
- escape.


---

## Combat System

Controls fighting behavior.


Examples:

- aiming;
- shooting;
- using cover.


# 3. Design Rule

AI behavior should be separated into independent systems.
