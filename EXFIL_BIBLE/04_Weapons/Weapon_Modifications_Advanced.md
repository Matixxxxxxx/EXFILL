# PROJECT EXFIL

# Advanced Weapon Modifications

Version: 0.1


# 1. Purpose

Advanced customization allows players to create unique weapons.


# 2. Modification Philosophy

Every modification changes something.

There are no free upgrades.


# 3. Modification Effects


A modification can improve:


## Performance

Example:

better recoil control.


## Handling

Example:

faster aiming.


## Tactical Ability

Example:

special optics.


## Appearance

Example:

skins and personalization.


# 4. Tradeoffs


Example:


Heavy barrel:

+
better accuracy

-
slower movement


Large magazine:

+
more ammunition

-
slower handling


# 5. Goal

Players should build weapons according to their strategy.
