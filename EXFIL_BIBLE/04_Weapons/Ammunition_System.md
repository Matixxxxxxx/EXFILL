# PROJECT EXFIL

# Ammunition System

Version: 0.1


# 1. Purpose

Ammunition is one of the most important parts of weapon balance.

In EXFIL, the weapon is only as effective as the ammunition it uses.


# 2. Core Philosophy

Players should think:

"Can I afford to use this ammunition?"


Not:

"The strongest ammo is always the correct choice."


# 3. Ammunition Characteristics


Every ammunition type has:


## Damage

How much damage it deals.


## Penetration

Ability to defeat armor.


## Availability

How difficult it is to obtain.


## Cost

Economic value.


## Special Properties

Additional effects.


# 4. Ammo Creates Choices


Example:


Cheap ammunition:

+
affordable

+
good for normal enemies

-
weak against armor


Expensive ammunition:

+
strong against armor

+
higher effectiveness

-
expensive risk


# 5. Design Goal

Ammunition should create strategic decisions before every raid.
