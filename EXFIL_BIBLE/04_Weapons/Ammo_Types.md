# Ammunition Types

Version: 0.1


# 1. Purpose

Different ammunition types create different tactical situations.


# 2. Main Categories


## Standard Ammunition

Role:

Basic everyday ammunition.


Advantages:

- cheap;
- available.


Disadvantages:

- lower armor penetration.


---

## Armor Piercing Ammunition

Role:

Defeating protected targets.


Advantages:

- high penetration.


Disadvantages:

- expensive;
- limited availability.


---

## Hollow Point Ammunition

Role:

Unarmored targets.


Advantages:

- high flesh damage.


Disadvantages:

- poor armor performance.


---

## Tracer Ammunition

Role:

Visual tracking.


Advantages:

- easier target tracking.


Disadvantages:

- reveals position.


---

## Special Ammunition

Role:

Rare tactical situations.


Examples:

- experimental rounds;
- limited production ammunition.


# 3. Balance Rule

No ammunition type should make others useless.
