# PROJECT EXFIL

# Weapon Condition System

Version: 0.1


# 1. Purpose

Weapon condition creates additional value and realism.


# 2. Condition Levels


## New

Maximum performance.


## Used

Normal performance.


## Damaged

Reduced effectiveness.


## Critical

Requires repair.


# 3. Condition Effects


Possible effects:

- accuracy;
- reliability;
- appearance;
- handling.


# 4. Design Rule

Condition should add gameplay depth.

It should not create constant frustration.


# 5. Emotional Value

A weapon with history should feel valuable.

Example:

"A rifle that survived 30 raids."
