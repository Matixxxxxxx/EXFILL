# PROJECT EXFIL

# Weapon Economy

Version: 0.1


# 1. Purpose

Weapons are part of the economic system.


The player should think:


"Can I afford to risk this weapon?"


# 2. Weapon Value


Value depends on:


- effectiveness;
- rarity;
- condition;
- demand;
- modifications.


# 3. Expensive Equipment

Should create:

- advantage;
- pressure;
- emotional risk.


# 4. Cheap Equipment

Should remain useful.


Budget weapons should allow:

- new players to compete;
- experienced players to take risks.


# 5. Goal

The economy should create choices.

Not force one optimal weapon.
