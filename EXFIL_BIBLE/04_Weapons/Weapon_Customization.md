# PROJECT EXFIL

# Weapon Customization

Version: 0.1


# 1. Purpose

Customization allows players to create personal weapons.


# 2. Modification Categories


Possible categories:


## Optics

Examples:

- red dots;
- scopes;
- thermal devices.


## Barrels

Effects:

- recoil;
- range;
- sound.


## Stocks

Effects:

- stability;
- handling.


## Grips

Effects:

- control;
- ergonomics.


## Magazines

Effects:

- capacity;
- reload speed.


## Attachments

Effects:

- tactical options.


# 3. Philosophy

Customization should create choices.

Not:

"Install every best attachment."


# 4. Goal

Two players can use the same weapon differently.
