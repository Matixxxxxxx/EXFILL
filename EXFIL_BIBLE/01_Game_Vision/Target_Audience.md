# PROJECT EXFIL

# Target Audience

Version: 0.1


# 1. Primary Audience

EXFIL is designed for players who enjoy:

- tactical shooters;
- extraction gameplay;
- competitive PvP;
- teamwork;
- progression systems;
- high-risk gameplay.


Main audience examples:

- Arena Breakout Infinite players;
- Escape from Tarkov players;
- Hunt: Showdown players;
- tactical FPS players.


# 2. Player Type

EXFIL targets several player types.


# Tactical Player

Enjoys:

- planning;
- positioning;
- strategy;
- careful decisions.


Motivation:

"Winning because I played smarter."


# Combat Player

Enjoys:

- gunfights;
- improving aim;
- defeating opponents.


Motivation:

"I want to become better."


# Collector Player

Enjoys:

- rare items;
- progression;
- equipment;
- achievements.


Motivation:

"I want to build something valuable."


# Team Player

Enjoys:

- cooperation;
- communication;
- helping teammates.


Motivation:

"My team created a great moment."


# 3. Age Target

Primary:

13+

Secondary:

Older players who enjoy deeper gameplay.


# 4. What Players Are Looking For

Players want:

- meaningful victories;
- exciting fights;
- progression;
- recognition;
- memorable experiences.


# 5. What Players Dislike

Avoid:

- pointless grinding;
- unfair advantages;
- boring repetition;
- empty worlds;
- meaningless rewards.
