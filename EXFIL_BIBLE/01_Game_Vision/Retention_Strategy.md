# Retention Strategy

Version: 0.1


# 1. Retention Philosophy

Players should return because they want to experience another raid.

Not because they are forced.


# 2. Short Term Retention

First sessions should provide:

- understandable gameplay;
- first success;
- exciting moments;
- clear goals.


# 3. Medium Term Retention

Players continue because of:

- better equipment;
- learning maps;
- improving combat skills;
- unlocking systems.


# 4. Long Term Retention

Players stay because of:

- mastery;
- rare achievements;
- reputation;
- personal stories;
- community.


# 5. Main Retention Question

After leaving the game:

The player should think:

"I want to see what happens next."
