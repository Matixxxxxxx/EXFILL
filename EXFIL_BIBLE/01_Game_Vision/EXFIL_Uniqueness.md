# PROJECT EXFIL

# EXFIL Uniqueness

Version: 0.1


# 1. Main Identity

EXFIL is not competing through:

- biggest amount of weapons;
- biggest maps;
- most complicated systems.


EXFIL competes through:

- memorable raids;
- meaningful choices;
- emotional moments.


# 2. Main Difference

Many extraction shooters focus on:

Loot → Money → Better Equipment


EXFIL focuses on:

Decision → Risk → Story → Progression


# 3. Core Question

Every raid should create situations players want to share.


Examples:

"How did you survive that?"

"How did you kill that team?"

"Why did you take that risk?"


# 4. Unique Experience

The player should feel:

"I created this moment."


Not:

"The game gave me another reward."
