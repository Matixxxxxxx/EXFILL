# Player Psychology

Version: 0.1


# 1. Why Players Return

Players return because of:


## Curiosity

"What will happen next raid?"


## Improvement

"I am getting better."


## Progression

"My character and equipment are growing."


## Stories

"I want another memorable moment."


# 2. Emotional Cycle

EXFIL should create this cycle:


Preparation

↓

Uncertainty

↓

Risk

↓

Challenge

↓

Decision

↓

Outcome

↓

Memory

↓

Return


# 3. Important Emotions


## Fear

Created by:

- losing equipment;
- unknown enemies;
- dangerous locations.


## Excitement

Created by:

- valuable discoveries;
- fights;
- successful escapes.


## Pride

Created by:

- difficult victories;
- teamwork;
- mastery.


# 4. Avoid Negative Emotions

Avoid:

Frustration without learning.

Boredom without purpose.

Grinding without reward.
