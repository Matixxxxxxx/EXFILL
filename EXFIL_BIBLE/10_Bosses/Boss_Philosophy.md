# Boss Philosophy

Version: 0.1


# 1. Purpose

Bosses create unique high-risk encounters.

They provide:

- challenge;
- rare rewards;
- memorable moments;
- world identity.


# 2. Core Philosophy

A boss is not just a stronger enemy.

A boss is:

- a character;
- a threat;
- a story.


# 3. Boss Encounter Goals


Every boss encounter should create:


## Preparation

Players need to decide:

- equipment;
- strategy;
- team.


## Risk

Failure should matter.


## Reward

Victory should feel valuable.


# 4. Avoid


Do not create:

- enemies with only increased health;
- random unfair attacks;
- mandatory farming.


# 5. Final Goal

Players should remember bosses.
