# PROJECT EXFIL

# Boss Balance Rules

Version: 0.1


# 1. Purpose

Bosses must be difficult but fair.


# 2. Rules


Boss should have:

- strengths;
- weaknesses;
- readable behavior.


# 3. Difficulty Sources


Good difficulty:

- tactics;
- positioning;
- preparation.


Bad difficulty:

- infinite health;
- impossible damage;
- cheating.


# 4. Final Rule

Players should feel:

"We lost because we made mistakes."

Not:

"The game was unfair."
