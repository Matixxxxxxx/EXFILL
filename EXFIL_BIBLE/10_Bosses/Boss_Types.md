# PROJECT EXFIL

# Boss Types

Version: 0.1


# 1. Purpose

Different bosses create different experiences.


# 2. Boss Categories


## Commander Boss


Role:

Military leader.


Features:

- tactical decisions;
- squad coordination;
- advanced equipment.


---

## Heavy Boss


Role:

Frontline threat.


Features:

- heavy armor;
- powerful weapons;
- aggressive behavior.


---

## Hunter Boss


Role:

Tracking specialist.


Features:

- stealth;
- traps;
- ambushes.


---

## Technical Boss


Role:

Specialist.


Features:

- gadgets;
- drones;
- electronic warfare.


---

## Mutated / Experimental Boss


Role:

Rare special threat.


Features:

- unique mechanics;
- unusual behavior.


# 3. Goal

Different bosses should require different strategies.
