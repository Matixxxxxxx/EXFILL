# PROJECT EXFIL

# Boss Spawn System

Version: 0.1


# 1. Purpose

Boss appearance creates uncertainty.


# 2. Spawn Philosophy


Bosses should not always appear.

Players should have:

- expectation;
- surprise.


# 3. Spawn Factors


Possible factors:


Location.


Server events.


Faction control.


Player activity.


# 4. Goal

Players should wonder:

"Is the boss here today?"
