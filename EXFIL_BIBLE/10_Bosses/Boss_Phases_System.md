# PROJECT EXFIL

# Boss Phases System

Version: 0.1


# 1. Purpose

Phases create dynamic fights.


# 2. Phase Example


Phase 1:

Normal combat.

Boss tests players.


Phase 2:

Boss changes tactics.

Uses new abilities.


Phase 3:

Critical state.

Maximum danger.


# 3. Requirements


Phase changes should be:

- understandable;
- meaningful;
- skill based.


# 4. Avoid


Do not create:

random transformations;

unfair surprises.
